ACS Tools
===============

ACS Command Center
******************

 Acs Command Center is a tool to start up, manager, ACS tools and containers 
 
It can be launched with:

.. code-block:: bash
   
   $ escs -start


JLOG
********************

jlog is a graphical ACS logging client. Manager must be up and running. It can be launched with: 
 
.. code-block:: bash
   
   $ jlog &
